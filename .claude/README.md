**Purpose:** Tell Claude HOW to work

- âœ… README.md - This documentation

---

### `business-context/` - For Claude.ai Project Only
**Purpose:** Business knowledge for SRS generation

**Contains:**
- Company information
- Target audience details
- Brand guidelines
- Technical constraints
- SRS template structure

**Used by:** Claude.ai Project to generate professional SRS documents

**NOT used by:** Claude Code (it reads the generated SRS, not this)

---

### `requirements/` - Generated SRS Documents
**Purpose:** What to build

**Contains:**
- SRS documents generated by Claude.ai Project
- Versioned (v1.0, v1.1, FINAL)
- Structured requirements with FR-XXX codes
- Acceptance criteria

**Used by:** Claude Code reads these to understand what to build

---

### `wireframes/` - Design Files
**Purpose:** How it should look (structure only)

**Contains:**
- HTML wireframes (one per page/feature)
- Can organize by feature, device, or flat structure

**Used by:** Claude Code extracts LAYOUT only (not styling)

---

### `src/` - Your Code
**Purpose:** What Claude Code builds

**Contains:**
- All your application code
- Components, pages, utilities, types
- Tests

**Modified by:** Claude Code creates and edits files here
